# Épicos

Unidade macro de trabalho. Protocolo completo em `../workflow.md`; template em `TEMPLATE.md`.

- Um arquivo por épico: `NNN-nome.md` (numeração sequencial).
- Estados: `proposto → escopado → aceito → em-execução → concluído`. O estado vive no próprio arquivo e só as skills do protocolo (ou Pedro) o mudam.
- DoD congela no aceite; mudança de DoD = novo loop de validação.
- Concluídos vão para `done/`.
